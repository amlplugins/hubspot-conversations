# @amlplugins/hubspot-conversations

Native `@hubspot/api-client` SDK re-exported under the `@amlplugins` namespace.

> HubSpot Conversations — inbox, threads, messages, channels, custom-channel webhooks.

## Install

```bash
npm install @amlplugins/hubspot-conversations
```

## Use

```ts
import { /* named exports from @hubspot/api-client */ } from "@amlplugins/hubspot-conversations";
// or
import sdk from "@amlplugins/hubspot-conversations";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
